Nmap Scan Reporter

Overview

Nmap Scan Reporter is a beginner cybersecurity project that demonstrates network reconnaissance and service enumeration using Nmap.

The project performs host scanning and service detection, then documents the findings in a report format.

Features
* Host discovery
* Open port identification
* Service detection
* Version detection
* Scan report generation

Tools Used
* Nmap
* Windows/Linux Command Line

Sample Scan Command:
nmap -sV <target-ip>

Example Output

22/tcp open ssh OpenSSH 8.9

80/tcp open http Apache 2.4.58

443/tcp open https Apache 2.4.58

Learning Outcomes

* Network reconnaissance
* Port scanning
* Service enumeration
* Reading Nmap results
* Basic vulnerability assessment workflow

This project is intended for educational purposes only. Scans should be performed only on systems you own or have permission to test.
